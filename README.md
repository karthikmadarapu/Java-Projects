Java Console Calculator
A lightweight, command-line interface (CLI) calculator built with Java. This program performs basic arithmetic operations and demonstrates clean code structure and input handling.
🚀 Features
Basic Arithmetic: Addition, Subtraction, Multiplication, and Division.
Error Handling: Prevents crashes from division by zero.
User Friendly: Simple prompts for number and operator input.
📋 Prerequisites
Java Development Kit (JDK): Version 8 or higher.
Terminal/Command Prompt: To run the application.
🛠️ How to Run
Clone or Download this repository to your local machine.
Navigate to the project folder in your terminal.
Compile the code:
bash
javac Calculator.java
Use code with caution.
Run the program:
bash
java Calculator
Use code with caution.

💻 Usage Example
text
Enter first number: 10
Enter an operator (+, -, *, /): *
Enter second number: 5
Result: 10 * 5 = 50
Use code with caution.
